# AI Financial Analytics Platform

## Features
- Gemini-powered AI SQL generation
- Financial insights (CFO-level)
- MySQL integration
- Dashboard + AI chat

## Run
pip install -r requirements.txt
python app.py

## Setup
- Add your Gemini API key
- Update DB connection

## Use Cases
- Revenue analysis
- Cost breakdown
- Margin insights
- Financial trends
